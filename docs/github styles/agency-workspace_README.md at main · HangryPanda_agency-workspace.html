      Desktop Workspace Application

      A browser-based desktop environment simulator built with React 19, TypeScript, and Vite. Features a multi-window interface with draggable/resizable applications, environment management, and a taskbar interface.

      Features

      • Desktop Environment: Simulated OS desktop with clickable icons and multi-window management
      • Window Management: Draggable, resizable application windows with minimize/maximize controls
      • Environment System: Switch between Development, Preview, and Production environments
      • Application Framework: Extensible system for building and deploying workspace applications
      • Taskbar Interface: Windows-style taskbar with active application indicators and system tray

      Getting Started

      Installation 

      npm install

      Development 

      npm run dev          # Start Vite dev server with HMR
npm run build        # Type check with tsc then build for production
npm run preview      # Preview production build locally
npm run lint         # Run ESLint on all files

      Using the Workspace

      Environment Switching

      The workspace supports three environments accessible via the taskbar:

      • Development (Blue badge): All applications visible, including experimental features
      • Preview (Orange badge): Beta and stable applications for testing
      • Production (Green badge): Only stable, released applications

      Click the environment badge in the taskbar to switch between environments. A watermark appears in Dev and Preview modes to prevent accidental use in production scenarios.

      Window Management

      • Open Applications: Click desktop icons to launch applications
      • Move Windows: Drag title bar to reposition windows
      • Resize Windows: Drag edges or corners to resize
      • Minimize: Click minimize button in title bar (window moves to taskbar)
      • Maximize: Click maximize button to fill screen
      • Close: Click X button in title bar
      • Focus: Click any window to bring it to front

      Desktop Interaction

      • Desktop Mode: Click the desktop background to bring desktop icons to the front
      • Window Mode: Click any window to bring all windows above desktop icons

      Built-In Applications

      Insurance Quote Summary Generator

      A comprehensive tool for generating insurance quote summaries with:

      • PDF quote upload and processing
      • Multi-policy support (Auto, Property, Boat, Fire, Life, Umbrella, Health, Personal Articles)
      • Live preview with real-time updates
      • Customizable header settings and client information
      • PDF export functionality

      For Developers

      Developer documentation is maintained in CLAUDE.md, including:

      • Architecture overview and design patterns
      • Creating new applications for the workspace
      • Z-index layering system
      • Environment system implementation
      • Scrolling patterns for fixed-height containers
      • Component structure and guidelines

      Tech Stack

      • React 19 with TypeScript 5.9
      • Vite 7.1 for build tooling
      • Tailwind CSS v4 for styling
      • Radix UI for accessible UI primitives
      • Lucide React for icons
